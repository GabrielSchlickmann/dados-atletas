# Projeto: Dados de Atletas #
#### Descrição ####

Este projeto implementa uma classe em JavaScript que define as informações e cálculos relacionados a atletas. Ele inclui funcionalidades para determinar a categoria etária, calcular o índice de massa corporal (IMC) e calcular uma média válida com base em notas de desempenho, descartando a maior e a menor nota.

O projeto também demonstra um exemplo de uso com dados de um atleta fictício, exibindo os resultados no console.
